# CleverPark

Android-застосунок для симуляції системи розумного паркування.

## Вимоги

- Пристрій на Android 8.0 або новіше
- Увімкнений інтернет для роботи з Firebase

## Встановлення

### Крок 1. Завантажте APK

Перейдіть у розділ [Releases](../../releases) цього репозиторію та завантажте файл `CleverPark.apk`.

### Крок 2. Дозвольте встановлення з невідомих джерел

Оскільки застосунок не розповсюджується через Google Play, необхідно дозволити встановлення з невідомих джерел:

1. Відкрийте **Налаштування** на вашому пристрої
2. Перейдіть у **Програми** або **Безпека** (залежно від пристрою)
3. Знайдіть пункт **Встановлення невідомих програм** або **Спеціальний доступ**
4. Виберіть файловий менеджер або браузер, через який відкриватимете APK
5. Увімкніть дозвіл **Дозволити з цього джерела**

### Крок 3. Встановіть застосунок

1. Відкрийте завантажений файл `CleverPark.apk` через файловий менеджер
2. Натисніть **Встановити**
3. Після завершення натисніть **Відкрити**

## Вхід у систему

Для використання застосунку необхідно зареєструватися або увійти за допомогою електронної пошти та пароля.

## Примітки

- Це тестова версія (debug build), призначена для демонстрації в рамках дипломної роботи.
- Для коректної роботи потрібне підключення до інтернету.
